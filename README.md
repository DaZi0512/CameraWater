# WebGL Camera Water Ripple

这是一个基于浏览器摄像头的实时水波纹交互效果项目。

项目通过调用设备摄像头获取实时画面，并使用WebGL1在视频画面上叠加水面折射、涟漪扩散、高光反射、RGB色散和透明水膜等视觉效果。整体效果类似手指在透明水面或液态玻璃表面划过，画面会随着指尖运动产生动态波纹。

项目核心使用FBO ping-pong双缓冲进行波动方程模拟，通过9-tap各向同性模板减少传统波纹模拟中常见的方形伪影，使涟漪扩散更加自然。渲染阶段结合坡度驱动折射、Laplacian曲率透镜、Fresnel反射、双瓣高光和波峰波谷明暗变化，增强水面包裹感和透明材质质感。

交互部分使用MediaPipe Hands进行手部关键点检测，只针对伸展状态下的指尖位置注入波纹，避免握拳、掌心或关节误触发。当前支持最多十个指尖同时触发水波效果，并支持前后摄像头切换，前置摄像头会自动镜像以符合自拍预览习惯。

项目提供两种视觉模式：

- **LIQUID**：偏柔和、透明、流体感更强的水面效果。
- **CRYSTAL**：偏锐利、高光更强、带有轻微晶体折射感的液态玻璃效果。

该项目适合用于实时互动视觉实验、浏览器端创意特效、WebGL图像处理演示、多模态交互展示以及摄像头实时滤镜效果验证。
